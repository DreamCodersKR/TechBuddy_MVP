## 1. 공통 - 레이아웃 시스템 & GNB

### 1-1. GNB (Global Navigation Bar)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  FLOWIT    │   커뮤니티     프로젝트   │      🔍 검색        │  로그인  회원가입  │
│  (로고)    │                         │                     │                  │
└─────────────────────────────────────────────────────────────────────────────┘
                                                            ↑ 비로그인 상태

┌─────────────────────────────────────────────────────────────────────────────┐
│  FLOWIT    │   커뮤니티     프로젝트   │      🔍 검색        │  🔔  [아바타 ▼]  │
└─────────────────────────────────────────────────────────────────────────────┘
                                                            ↑ 로그인 상태

[아바타 ▼] 클릭 시 드롭다운:
┌──────────────┐
│  닉네임      │
│  Lv.5 ⚡    │
│  ─────────  │
│  마이페이지  │
│  내 게시글  │
│  북마크     │
│  설정       │
│  ─────────  │
│  로그아웃   │
└──────────────┘
```

**스펙 메모**
- 높이: 56px 고정
- 로고 클릭 → `/`
- `커뮤니티` 호버 시 서브탭 노출 (게시판 | 아고라 | 팀원모집)
- `프로젝트` 클릭 → `/workspaces`
- 알림(🔔) 클릭 → 알림 드롭다운 패널 (별도 설계)

### 1-2. 커뮤니티 서브탭 (GNB 아래)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  GNB...                                                                     │
└─────────────────────────────────────────────────────────────────────────────┘
┌─────────────────────────────────────────────────────────────────────────────┐
│           게시판 [선택]   │   아고라   │   팀원모집                           │
│           ─────────────────────────────                                    │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 1-3. PM툴 레이아웃 (좌측 사이드바)

```
┌──────────────┬──────────────────────────────────────────────────────────────┐
│ FLOWIT  [≡]  │                                                              │
│ ──────────── │                메인 컨텐츠 영역                               │
│ 📋 내 Task   │                                                              │
│ ──────────── │                (리스트뷰 / 칸반보드 / Sprint 등)              │
│ 내 워크스페이스 ▼│                                                          │
│  📁 포트폴리오 │                                                            │
│  📚 알고리즘  │                                                              │
│  + 새로 만들기│                                                              │
│ ──────────── │                                                              │
│ [선택됨]     │                                                              │
│  포트폴리오 ▼ │                                                              │
│   └ Task     │                                                              │
│   └ Sprint   │                                                              │
│   └ 칸반보드  │                                                              │
│   └ 채팅 🔒  │    ← 스터디룸만, 준비중                                       │
│   └ 멤버     │                                                              │
│   └ 설정     │                                                              │
│ ──────────── │                                                              │
│ 💬 AI 멘토링  │    ← 하단 고정                                               │
└──────────────┴──────────────────────────────────────────────────────────────┘
```

**스펙 메모**
- 사이드바 너비: 240px
- [≡] 클릭 시 사이드바 접기/펼치기
- 채팅 메뉴: 스터디룸 타입일 때만 노출, `🔒 준비중` 배지 표시
- AI 멘토링 버튼: 사이드바 하단 고정
