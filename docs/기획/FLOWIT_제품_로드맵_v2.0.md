### M9: 이메일 자동화 & 온보딩 (2026-05-31)
**목표**: 이탈 방어 자동화, 첫 경험 최적화

**필요 기능**:
- Resend 이메일 서비스 연동 (`resend.com`)
- 웰컴 이메일 (가입 즉시)
- D+1: "첫 번째 AI 멘토링 해보셨나요?" 온보딩 넛지
- D+3: "아고라에서 질문 하나 올려보세요 + 10cr 증정"
- D+7: "이번 주 인기 글 요약 + TIL 챌린지 초대"
- 비활성 유저 (14일 미접속) 리인게이지 이메일
- 크레딧 10개 미만 → 경고 이메일
- 신규 가입자 인터랙티브 온보딩 (5단계 UI 투어)

**이메일 트리거 목록**:
```
WELCOME           → 가입 즉시
ONBOARDING_D1     → 가입 +1일
ONBOARDING_D3     → 가입 +3일
ONBOARDING_D7     → 가입 +7일
INACTIVE_14D      → 14일 미접속
LOW_CREDIT        → 크레딧 < 10
WEEKLY_DIGEST     → 매주 금요일 (인기글 요약)
INQUIRY_REPLY     → 문의 답변 완료
```

**성공 지표**: 7일 리텐션 40%↑, 이메일 오픈율 30%↑

### M10: 리텐션 & 게이미피케이션 (2026-06-30)
**목표**: 매일 접속할 이유 → DAU 극대화

**TIL (Today I Learned) 시스템**:

- GitHub 잔디처럼 년간 기록 시각화
- 연속 기록 스트릭 (3일/7일/30일 마일스톤 → 크레딧/XP 보상)
- 공개/비공개 설정, 좋아요/댓글

**데일리 퀘스트**:
```
오늘의 퀘스트 목록 예시:
□ TIL 작성하기           +5cr
□ 아고라 답변 1개 달기    +3cr
□ AI 멘토링 1회 이용      +2cr (사용 여부만)
□ 팀 태스크 1개 완료      +5cr
```

**바이럴 루프**:
- 친구 초대 링크 발급 (초대자 +50cr, 피초대자 +30cr)
- 잔디/뱃지 SNS 공유 기능
- 주간 TOP 기여자 랭킹 (커뮤니티 내 공개)

**성공 지표**: DAU/MAU 25%↑, 주간 세션 3회↑

### M12: AI 고도화 (Context-Aware + RAG) (2026-08-31)
**목표**: "맥락을 아는 AI" → 노션AI/챗GPT 대비 압도적 차별화

**Context-Aware AI 멘토**:
```
현재 보고 있는 화면을 AI가 인식:
- 칸반 보드 → "백로그에 있는 태스크 기반으로 다음 스프린트 계획 짜줘"
- Task 상세 → "이 태스크 구현하는 데 참고할 코드 패턴 알려줘"
- 산출문서 → "이 ERD에서 N+1 문제 발생 가능한 쿼리 찾아줘"
```

**RAG 파이프라인**:
```
사용자 문서 업로드
→ Supabase pgvector 임베딩 저장
→ AI 질문 시 관련 문서 자동 검색
→ 컨텍스트에 주입 → 정확한 답변
```

**AI 커리큘럼 추천**:
- 사용 기술 스택 + AI 멘토링 질문 패턴 분석
- "다음으로 공부하면 좋을 것" 주 1회 추천
- 부트캠프 수료 후 취업까지 개인화 로드맵

**성공 지표**: AI 세션 만족도 4.0↑, 크레딧 소비 50%↑

**성공 지표**: 월 커피챗 50건↑, 크레딧 소비 2배↑

### M14: B2B 부트캠프 파트너십 (2026-11-30)
**목표**: 부트캠프 기관 B2B SaaS → MRR 안정화

**영업 타겟 우선순위**:
1. 스파르타코딩클럽 (규모 크고, 커뮤니티 친화적)
2. 엘리스 (AI/취업 중심, FLOWIT과 방향 일치)
3. 코드스테이츠 (부트캠프 선두, 팀프로젝트 강조)
4. 멀티캠퍼스 (삼성 계열, 대규모)
5. 패스트캠퍼스

**B2B 가격 정책**:
```
Basic  (50명 이하)  → 월 30만원
Pro    (200명 이하) → 월 80만원
Enterprise (무제한) → 협의
```

**강사 대시보드 기능**:
- 수강생 활동량 그래프 (AI 멘토링, 칸반 완료율)
- "어떤 기술에서 가장 많이 막히나?" 질문 트렌드 분석
- 이탈 위험 수강생 예측 (7일 미접속 알림)
- 파이널 프로젝트 일괄 워크스페이스 생성

## 6. 경쟁 차별화 포지셔닝

```
          │ PM툴
          │ (Linear/Notion)
          │         ← FLOWIT →
   커뮤니티 ─────────────────────── AI 멘토링
  (Disquiet)│                        (ChatGPT)
          │
          │ 채용/취업
            (원티드/점핏)
```

**FLOWIT만의 Moat:**
1. **맥락 연결성**: PM툴 + 커뮤니티 + AI가 하나의 데이터로 연결
2. **활동의 증명**: 잔디 + Verified Project = 포트폴리오 자동 생성
3. **주니어 특화 AI**: 부트캠프 수료생 니즈에 최적화된 멘토링
4. **크레딧 생태계**: 학습(소비) ↔ 기여(획득)의 선순환
