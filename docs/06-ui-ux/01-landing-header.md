# 랜딩 페이지 - 헤더 가이드

## 개요

TechBuddy 랜딩 페이지(techbuddy.com)의 **헤더** 디자인 가이드입니다.
UI/UX 레퍼런스는 [Linear.app](https://linear.app)입니다.

---

## 헤더 구성 (MVP)

### 레이아웃

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│  [Logo]     커뮤니티  |  프로젝트  |  멘토링(Coming Soon)  |  소개    [시작하기]  │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### 메뉴 상세

| 메뉴 | 동작 | 설명 |
|------|------|------|
| **Logo** | 클릭 → 홈(최상단) | TechBuddy 로고 |
| **커뮤니티** | 클릭 → 섹션 스크롤 또는 별도 페이지 | 커뮤니티 기능 소개 |
| **프로젝트** | 클릭 → 섹션 스크롤 또는 별도 페이지 | 프로젝트 팀빌딩 기능 소개 |
| **멘토링** | 비활성 상태 + "Coming Soon" 뱃지 | Phase 3 예고 |
| **소개** | 클릭 → 소개 페이지 | TechBuddy 비전 소개 |
| **시작하기** | 클릭 → 회원가입/로그인 | CTA 버튼 (강조색) |

### 디자인 노트

- **스타일**: Linear.app 참고 (다크 테마, 미니멀)
- **폰트**: 깔끔한 산세리프
- **CTA 버튼**: 대비되는 색상으로 강조 (예: 파란색, 보라색)
- **반응형**: 모바일에서는 햄버거 메뉴로 변환

## 향후 확장 (유저 증가 시)

```
┌────────────────────────────────────────────────────────────────────────────┐
│  [Logo]   기능 ▼  |  인사이트  |  소개  |  문의    [로그인] [시작하기]       │
└────────────────────────────────────────────────────────────────────────────┘
                │
                ├─ 커뮤니티
                ├─ 프로젝트
                └─ 멘토링
```

### 확장 시 추가 메뉴

| 메뉴 | 설명 |
|------|------|
| **기능 (드롭다운)** | hover 시 fade-in 애니메이션으로 하위 메뉴 표시 |
| **인사이트** | 블로그, 취준 팁, 부트캠프 후기 등 콘텐츠 |
| **문의** | Contact, FAQ |
