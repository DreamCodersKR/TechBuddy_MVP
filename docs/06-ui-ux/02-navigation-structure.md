# FLOWIT 네비게이션 구조 가이드

## 개요

FLOWIT의 **네비게이션 구조** 가이드입니다.
v1.4에서 사이드바 기반에서 **상단 네비게이션 기반**으로 변경되었습니다.

---

## 네비게이션 변경 이력

| 버전 | 구조 | 스타일 |
|------|------|--------|
| v1.3 | 사이드바 네비게이션 | Linear 스타일 |
| **v1.4** | **상단 네비게이션** | **커뮤니티 사이트 스타일** |

---

## 상단 네비게이션 바

### 레이아웃

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                                                                                 │
│  [Logo]     플랫폼  |  커뮤니티  |  Q&A  |  팀원 모집      [검색]    [로그인]    │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
```

### 메뉴 상세

| 메뉴 | 경로 | 설명 | 상태 |
|------|------|------|------|
| **Logo** | `/` | FLOWIT 로고, 클릭 시 홈으로 | 활성 |
| **플랫폼** | `/platform` | 서비스 소개, 공지사항 | 활성 |
| **커뮤니티** | `/community` | 자유게시판, 취업/진로 등 | 활성 |
| **Q&A** | `/qna` | 기술 Q&A, 취업 Q&A | 활성 |
| **팀원 모집** | `/recruit` | 프로젝트 팀원 모집 | 활성 |
| **검색** | - | 통합 검색 | 활성 |
| **로그인** | `/auth/login` | 로그인/회원가입 | 활성 |

### 로그인 후 변경

```
┌─────────────────────────────────────────────────────────────────────────────────┐

│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
                                                                      │
                                                          ┌───────────┴───────────┐

                                                          │  ─────────────────    │
                                                          │  로그아웃             │
                                                          └───────────────────────┘
```

## 카테고리 구조

### 커뮤니티 (/community)

| 카테고리 | slug | 설명 |
|----------|------|------|
| 전체 | `all` | 전체 게시글 |
| 자유 | `free` | 자유로운 주제 |
| 취업/진로 | `career` | 취업 정보, 진로 상담 |
| AI | `ai` | AI 관련 토론 |
| 기술 | `tech` | 기술 관련 정보 공유 |

### Q&A (/qna)

| 카테고리 | slug | 설명 |
|----------|------|------|
| 전체 | `all` | 전체 Q&A |
| 기술 | `tech` | 기술 관련 질문 |
| 취업/진로 | `career` | 취업, 면접 질문 |
| 기타 | `etc` | 기타 질문 |

### 팀원 모집 (/recruit)

| 카테고리 | slug | 설명 |
|----------|------|------|
| 전체 | `all` | 전체 모집글 |
| 사이드 프로젝트 | `side-project` | 개인 프로젝트 |
| 스터디 | `study` | 스터디 모집 |
| 해커톤 | `hackathon` | 해커톤 팀 모집 |

## 디자인 가이드라인

### 스타일

- **배경색**: 흰색 (#FFFFFF)
- **텍스트**: 다크 그레이 (#1F2937)
- **활성 메뉴**: Primary 색상 (추후 확정)
- **구분선**: 연한 그레이 (#E5E7EB)

### 반응형 브레이크포인트

| 사이즈 | 너비 | 네비게이션 |
|--------|------|-----------|
| Mobile | < 768px | 햄버거 메뉴 |
| Tablet | 768px - 1024px | 축소된 상단 네비 |
| Desktop | > 1024px | 전체 상단 네비 |
