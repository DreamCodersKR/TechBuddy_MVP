# FLOWIT 서비스 플로우 가이드

## 개요

FLOWIT의 **서비스 플로우** 및 **사용자 여정** 가이드입니다.
v1.4에서 랜딩 페이지 기반에서 **커뮤니티 베이스**로 변경되었습니다.

---

## 플로우 변경 이력

| 버전 | 플로우 | 특징 |
|------|--------|------|
| v1.3 | 랜딩 → 시작하기 → 앱 | Linear 스타일, SaaS형 |
| **v1.4** | **바로 커뮤니티 메인** | **커뮤니티 사이트 스타일** |

---

## 서비스 플로우

### 1. 메인 진입 플로우

```
사용자 방문 (flowit.com)
         │
         ▼
┌─────────────────────────┐
│   메인 페이지 (/)        │
│   = 커뮤니티 홈          │
│                         │
│   - 실시간 인기글        │
│   - 실시간 인기 Q&A      │
│   - 팀원 모집 카드       │
└─────────────────────────┘
         │
         ├─── 콘텐츠 열람 (비로그인 가능)
         │
         └─── 작성/참여 시도
                  │
                  ▼
         ┌───────────────┐
         │  로그인 유도   │
         │  모달/페이지   │
         └───────────────┘
```

│                                                             │

```

### 3. 콘텐츠 이용 플로우

```
┌─────────────────────────────────────────────────────────────┐
│                    커뮤니티 이용 플로우                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   [메인 페이지]                                              │
│        │                                                    │
│        ├─── [커뮤니티] 클릭                                  │
│        │         │                                          │
│        │         ▼                                          │
│        │    /community                                      │
│        │    게시글 목록 (자유, 취업/진로, AI 등)              │
│        │         │                                          │
│        │         ├─── 게시글 클릭 → /community/[id]          │

│        │                                                    │
│        ├─── [Q&A] 클릭                                       │
│        │         │                                          │
│        │         ▼                                          │
│        │    /qna                                            │

│        │                                                    │

│                  │                                          │

│             팀원 모집 목록                                    │
│                                                             │

## 접근 권한

### 비로그인 vs 로그인

| 기능 | 비로그인 | 로그인 |
|------|:-------:|:------:|
| 메인 페이지 열람 | O | O |
| 게시글 목록 열람 | O | O |
| 게시글 상세 열람 | O | O |
| Q&A 열람 | O | O |
| 팀원 모집 열람 | O | O |
| **게시글 작성** | X | O |
| **댓글 작성** | X | O |
| **좋아요/북마크** | X | O |
| **프로젝트 참여** | X | O |
| **마이페이지** | X | O |

### 로그인 유도 시점

비로그인 상태에서 다음 액션 시 로그인 모달 표시:
- 글쓰기 버튼 클릭
- 댓글 작성 시도
- 좋아요/북마크 클릭
- 프로젝트 생성/참여
