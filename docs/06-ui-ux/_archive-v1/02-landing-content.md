## 기능 소개 섹션

각 섹션은 **타이틀 + 설명 + 스크린샷/이미지**로 구성.
레이아웃: 텍스트와 이미지를 번갈아 배치 (좌-우, 우-좌).

### 1. 커뮤니티

| 요소 | 내용 |
|------|------|
| **타이틀** | 커뮤니티 |
| **설명** | 인사이트를 나누고, 네트워크를 넓히세요 |
| **이미지** | 앱 스크린샷 (개발 후 추가) |

### 2. 프로젝트

| 요소 | 내용 |
|------|------|
| **타이틀** | 프로젝트 |
| **설명** | 팀을 구성하고 함께 만들어가세요 |
| **이미지** | 앱 스크린샷 (개발 후 추가) |

### 3. 멘토링 (Coming Soon)

| 요소 | 내용 |
|------|------|
| **타이틀** | 멘토링 |
| **뱃지** | Coming Soon |
| **설명** | 현직자에게 조언을 받아보세요 |
| **이미지** | 목업 또는 일러스트 |
